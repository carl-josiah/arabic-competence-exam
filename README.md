# arabic-competence-exam

A website that tests the user's knowledge and ability about the Arabic Language.

# root PHP files

The exam.php, results.php, and index.php are the main .php pages that the user visits. These act as the _Views_ in the MVC.

# Architectural Patterns

- Model-View-Controller (MVC)

# Principles and Patterns

- Separation of Concerns (SoC)
- Single Source of Truth Pattern

# Industry Standards

- PSR-1 and PSR-12 (PHP Standards)
- Using JSON for data exchange
- Stateless API
